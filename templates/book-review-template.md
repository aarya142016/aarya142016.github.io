**How to use this template:**

1. Copy this file into the `src/content/book-reviews/` directory
2. Rename it to something like `my-book-review.md` (use lowercase and hyphens)
3. Fill in all the fields in the frontmatter (the part between ---)
4. Save the file and it will automatically appear on the Books page!

**Required fields:**
- title: The book's title
- author: The book's author
- yearRead: Year you read the book
- rating: 1-5 stars
- summary: 2-3 sentences about the book
- learned: List of 3 things you learned
- favoritePart: Your favorite part of the book
- recommend: Your recommendation

**Optional fields:**
- genre: The book's genre
- tags: Array of tags for categorization
- dateRead: When you finished reading (YYYY-MM-DD format)

**Rating Guide:**
- ⭐ (1 star) - Didn't enjoy it
- ⭐⭐ (2 stars) - Not great
- ⭐⭐⭐ (3 stars) - It was okay
- ⭐⭐⭐⭐ (4 stars) - Really good!
- ⭐⭐⭐⭐⭐ (5 stars) - Amazing! Loved it!
