The Creative Problem Solving book taught me a lot and is very handy for puzzle-type competitions. I learned how to read problems carefully and reflect on the problems I solved. Later in the book, I learned many different problem-solving strategies: drawing a picture, making a table, writing an equation, etc.

One thing I noticed about the problems in this book is that it includes problems that appeared in math competitions, so it's a great book if you're preparing for competitions. I also learned all the topics in problem solving that the book taught me: Number Patterns, Divisibility, Cryptarithms (really tough), etc. 

What I really liked about the book is the author's way of writing. He explains everything very clearly and always has a simple example, which is very unique. This helped me change my way of thinking. At first, I used to think that if you solve the problem and get it right, it's over—but you're supposed to reflect on your problem so you remember the steps and can use them in similar problems.

Another reason I liked this book is that it starts from the very beginning—no knowledge of problem solving is needed.

I think the author could make the book longer. It was only 250 pages. Maybe add more problem-solving strategies or more topics. It only took me 3 months to read through it once.
